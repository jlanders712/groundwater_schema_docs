# Table: Aquifer Predictions

## Overview

- Name: aquifer_predictions
- Type: Related attribute table
- Description: DNRC machine learning model aquifer predictions, including DNRC source aquifer and DNRC aquifer code.
- Primary key: GUID
- Parent/child role: Child layer.
